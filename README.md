# Site Classifier

Десктопное приложение для автоматической классификации русскоязычных сайтов по тематике. Принимает один URL или CSV-файл со списком сайтов, парсит содержимое через Selenium и возвращает предсказанную категорию.

## Как это работает

Selenium запускает Chrome в headless-режиме с отключёнными изображениями и JS, обходит сайт на глубину 1 уровня и собирает текст. Параллельно запускается до 7 процессов через joblib. Собранный текст проходит предобработку: убирается всё нерусское, текст лемматизируется через pymystem3, отфильтровываются стоп-слова.

Классификатор: TF-IDF + логистическая регрессия. Модели сериализованы в joblib и лежат в `src/classifer_model/saves_models/`.

## Режимы работы

**Одиночный URL** - вводишь адрес вручную, получаешь категорию.

**CSV с URL** - загружаешь файл со списком адресов (один столбец, разделитель `;`), приложение классифицирует все сайты пачкой и показывает таблицу. Результат можно сохранить в CSV.

**Тестовый режим** - загружаешь CSV с двумя столбцами: URL и правильная категория. После классификации таблица красит ячейки: зелёный - совпало, красный - нет. Удобно для проверки качества модели на новой выборке.

## Установка

```bash
pip install -r requirements.txt
```

Дополнительно нужен ChromeDriver, совместимый с установленной версией Chrome. Положить `chromedriver` в `src/`.

## Запуск

```bash
cd src
python menu.py
```

## Формат входных CSV

Файл для пакетной классификации:
```
site
https://example.ru
https://another.ru
```

Файл для тестового режима:
```
site;label
https://example.ru;Строительство
https://news.ru;Новости
```

Разделитель везде - точка с запятой.

## Зависимости

| Пакет | Назначение |
|---|---|

| scikit-learn | TF-IDF, логистическая регрессия |
| pymystem3 | Лемматизация русского текста |
| selenium | Парсинг сайтов |
| beautifulsoup4 | Разбор HTML |
| joblib | Сериализация моделей, параллелизм |
| aiohttp | Альтернативный async-скрапер |
| yarl | Работа с URL |
